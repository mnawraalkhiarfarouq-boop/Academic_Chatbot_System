# Academic Chatbot System 🎓🤖
**نظام محادثة ذكي لدعم الخدمات الأكاديمية - جامعة أم درمان الإسلامية**

## 📝 نبذة عن المشروع
نظام محادثة ذكي مدعوم بالذكاء الاصطناعي يهدف إلى تحسين جودة الخدمات الأكاديمية وتسهيل وصول الطلاب للمعلومات داخل **كلية علوم الحاسوب وتقانة المعلومات**.

## ⚠️ مشكلة البحث
يواجه الطلاب والمؤسسات التعليمية تحديات تشمل:
* **بطء الاستجابة:** خاصة في فترات التسجيل والامتحانات.
* **الضغط الإداري:** كثرة الطلبات الروتينية التي تستهلك جهد الموظفين.
* **تأخر المعلومة:** الاعتماد على الطرق التقليدية والورقية.
* **الفجوة اللغوية:** غياب أنظمة ذكية تدعم اللغة العربية والسياق المحلي للجامعات السودانية.

## 🎯 أهداف المشروع وفرضياته
يسعى المشروع لإثبات أن استخدام نظام محادثة آلي يؤدي إلى:
1. تقليل الوقت والجهد في الحصول على المعلومة الأكاديمية.
2. تحسين التواصل بين الطالب والإدارة.
3. رفع مستوى رضا الطلاب عبر دعم اللغة العربية.
4. تعزيز التحول الرقمي داخل الكلية.

## 🛠 التقنيات المستخدمة (Tech Stack)
* **اللغة الأساسية:** Python
* **إطار العمل:** Flask (Backend)
* **الواجهة الأمامية:** HTML5, CSS3, JavaScript (Fetch API)
* **الذكاء الاصطناعي:** معالجة اللغة الطبيعية (NLP)
* **إدارة النسخ:** Git & GitHub

## 📂 هيكلة المشروع
- `app.py`: الملف الأساسي لتشغيل السيرفر والربط مع نموذج الذكاء الاصطناعي.
- `templates/`: يحتوي على واجهة الشات (`index.html`).
- `static/`: ملفات التنسيق والرسوم.
- `models/`: ملفات معالجة اللغة الطبيعية والبيانات الأكاديمية.

## 📍 حدود البحث
* **المكان:** كلية علوم الحاسوب وتقانة المعلومات - جامعة أم درمان الإسلامية.
* **الزمان:** العام الأكاديمي 2025م - 2026م.
