تطوير نظام محادثة ذكي (Chatbot) لتحسين الخدمات الأكاديمية
​بالتطبيق على كلية علوم الحاسوب وتقانة المعلومات - جامعة أم درمان الإسلامية
​📝 نبذة عن المشروع
​يهدف هذا المشروع إلى ردم الفجوة بين الطالب والإدارة عبر تطوير نظام محادثة ذكي يفهم السياق الأكاديمي باستخدام تقنيات معالجة اللغات الطبيعية (NLP). يوفر النظام قناة تواصل ذكية تعمل على مدار الساعة للإجابة على استفسارات الطلاب المتكررة حول الجداول، النتائج، واللوائح الأكاديمية.  
​⚠️ مشكلة البحث
​يأتي هذا النظام لمعالجة عدة تحديات تواجه الكلية، منها:  
​بطء الاستجابة للاستفسارات خاصة في فترات التسجيل والامتحانات.  
​الضغط الكبير على الموظفين الإداريين نتيجة الطلبات الروتينية.  
​غياب قناة موحدة وذكية تدعم اللغة العربية والسياق المحلي للجامعات السودانية.  
​🚀 الأهداف الرئيسية
​أتمتة الإجراءات: تقديم دعم فني وخدمات إلكترونية بشكل أسرع.  
​تحسين تجربة الطالب: توفير إجابات فورية ودقيقة على مدار الساعة.  
​دعم اللغة العربية: فهم المصطلحات الأكاديمية المحلية والدارجة داخل الحرم الجامعي.  
​🛠️ التقنيات المستخدمة
​بناءً على الجدوى التقنية المخطط لها:  
​اللغات: Python (المحرك الأساسي للذكاء الاصطناعي).  
​إطار العمل: Flask / FastAPI (لأنظمة الويب).
​قاعدة البيانات: MySQL / SQLite لتخزين الجداول والنتائج.  
​الذكاء الاصطناعي: نماذج المحولات (Transformers) وتقنيات NLP.
